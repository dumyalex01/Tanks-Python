Descriere Joc:
   Relieful este construit cu ajutorul desenarii pe harta de joc a unor dreptunghiuri care sa favorizeze implementarea unui "deal".
Sunt utilizate sunete, functiile fiind preluate de pe geeksforgeeks, din tutorialul prezentat la curs.
Punctele de viata ale tank-urilor au valoarea 100
Damage-ul dat atunci cand este activat proiectilul de urmarire este 10.
Damage-ul standard este de 20
Damage-ul dat atunci cand este activat proiectilul de suprafata mica este 40(poate fi folosit de 2 ori).
Conditia ca proiectulul de suprafata mica SA LOVEASCA tank-ul este ca acesta sa-i atinga plafonul(partea de sus)
Traiectoria este  construita astfel incat sa existe sanse cat mai mari de acoperire a tuturor punctelor din harta


Dificultati in implementare:
Gasirea unei ecuatii pentru a acoperi toata mapa.
Distrugerea reliefului.
